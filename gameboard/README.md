# Gameboard - jQuery DOM interaction practice
> A website that simulates an app that displays NBS scores.

## General info
The website is deployed with [Surge](https://surge.sh/). 
It can be viewed at: [Gameboard](http://false-friends.surge.sh/)


## Technologies
* JavaScript
* HTML
* CSS
* jQuery v3.3.1
* Bootstrap v3.2.0


## Features
* User can click the 'More' link to display a dropdown menu with options
* In the 'More' dropdown menu the user can click the 'Share' link to display the share menu
* When the user clicks on the notification bell it turns yellow
* Use .toggle() to show and hide elements
* Use 'event.currentTarget' to target the specific element clicked
* Use .siblings() and .next() to target DOM elements
